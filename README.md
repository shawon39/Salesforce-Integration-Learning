# Salesforce Integration Learning

A complete, hands-on handbook for learning Salesforce integration — fundamentals, authentication, inbound/outbound APIs, event-driven patterns, bulk/async processing, modern APIs, security, tooling, and projects.

## 📖 Read the Handbook

The full handbook is published with GitHub Pages:

**https://shawon39.github.io/Salesforce-Integration-Learning/**

## Contents

- **00-START-HERE** — orientation and learning path
- **01-Fundamentals** — core concepts and vocabulary
- **02-Integration-Patterns** — integration patterns
- **03-Authentication** — OAuth flows, JWT, named credentials
- **04-Inbound-APIs** — REST/SOAP/Apex web services
- **05-Outbound-Callouts** — callouts from Salesforce
- **06-Event-Driven** — Platform Events, CDC, streaming
- **07-Bulk-Async** — Bulk API, Batch, Queueable
- **08-Modern-APIs** — GraphQL, Connect, UI API
- **09-Security-Limits** — governor limits and security
- **10-Tools-Middleware** — integration tooling
- **11-Hands-On-Projects** — practical projects
- **12-Cheatsheets** — quick references
- **13-Whats-New-2026** — latest platform updates

The single-file handbook (`Salesforce-Integration-COMPLETE-Handbook.html`) bundles everything into one page; `index.html` is a copy served as the GitHub Pages landing page.
